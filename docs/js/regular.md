## 正则

```js
简写://
实例化:new RegExp(‘字符串’,’修饰符’)

^ 开头
$ 结尾

[\u4e00-\u9fa5] 中文区间范围
\n 换行
\d 一个数字
\D 一个非数字，（不是数字）
修饰符：
g - > global 全局
i - > ignore 忽略

\ 转义符
**需要注意的是在写 / 记得转义

当\ + 字母的时候会有特殊含义，这种带有特殊含义的，我们叫元字符

exec [ɪɡˈzek]
/规则/.exec(字符串)
找到正则中第一个匹配的字符且放到数组中。

test[test]
/规则/.test(字符串)
检测正则是否匹配字符串，成立就返回true，否则false
```
